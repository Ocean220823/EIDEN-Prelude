🗡️ EIDEN — Only the Blade Remains

“Only the blade remembers what the world has forgotten.”

A third-person action game showcasing an AI-controlled antagonist system with adaptive combat behavior, built using Unreal Engine 5.

🎮 Project Overview

EIDEN is set on a cursed island where a failed immortality ritual has unleashed seven immortal demons.

You play as Ryouma, an immortal ronin whose only path to death is defeating these demons and absorbing their essence.

This project focuses on demonstrating Artificial Intelligence in games, specifically an enemy system that reacts dynamically to the player.

🎯 Key Features

⚔️ Third-person combat system

🧠 Intelligent AI enemy behavior

👁️ AI perception (vision-based detection)

🔁 Finite State Machine (FSM)

🌳 Behavior Tree system

📊 Blackboard-based AI memory

🎲 Dynamic combat decision making

🔥 Enraged mode based on health

🚶 NavMesh pathfinding

🧠 AI System (Core Focus)

The enemy AI uses a hybrid architecture:

🔹 Finite State Machine (FSM)

Controls high-level states:

Idle

Patrol

Alert

Chase

Combat

Enraged

Dead

🔹 Behavior Tree

Handles:

Attack selection

Dodge logic

Reaction timing

🔹 Blackboard

Stores:

Player position

Distance

Health

Aggression level

🔹 Adaptive Combat System

Adjusts behavior based on player actions

Prevents repetitive combat

Creates dynamic encounters
